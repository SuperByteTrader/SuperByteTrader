## 什么是SuperByteTrader
- SuperByteTrader是一个基于SpringCloud/SpringBoot, Rocketmq, Redis, MySQL, Apollo, Euraka等技术框架实现的一套含订单系统/交易撮合/成交结算/行情计算/实时推送等功能的软件系统，以及量化交易平台的一整套交易软件解决方案。
- 在如下硬件资源下，性能指标数据如下:
  - 下单延迟
  - 交易撮合性能
  - 结算处理能力
  - 行情计算处理能力
  - 实时推送延迟
- SuperByteTrader实现功能
  - 注册
  - 登录
  - 现货交易
  - 撮合
  - 成交结算
  - 行情
  - websocket推送
